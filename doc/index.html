DockerHubとは？

Dockerのイメージの公開先
  • 個人や団体、企業が作ったimageの集約先
  • githubをリスペクしているので、思想は近い＝用語が似てる
dockerhub

DockerHubの注意事項

  • Docker Certified(Docker社が認証した)imageとそれ以外
  • Docker Certified：有名どころの団体でそのまま使っても安心(OracleやIBM製品)
  • 非Docker Certified：上記以外個人が作ったimageでたとえばスパイウェア的なものが含まれていても、image化されているので気づきにくい
  • 非Docker Certified：非Docker CertifiedだけどOFFCIAL IMAGEのタグが付いているもの(ubuntuやPostgres)

Dockerのセキュリティー回りの事案

  • 露出したDocker APIを介してコンテナに侵入するボット型マルウェア「AESDDoS」について解説
  • コンテナランタイム「runc」に脆弱性、Dockerなどに影響 - JPCERT/CC
  • 「Docker Hub」ポータル、著名コンテナの20％に設定ミス
    • 便利そうなimageがあってもすぐに飛びつかない。
    DockerCertified,OFFICAL IMAGE,他Reviewや☆Ratingなどに注意する
